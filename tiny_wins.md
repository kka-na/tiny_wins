# 💫 Tiny Wins
**작은 성공들이 모여 큰 변화를 만드는 35일 습관 형성 웹 서비스**

---

## 📋 프로젝트 개요

### 핵심 콘셉트
- **과학적 기반**: 뇌의 S-R(자극-반응) 시스템과 R-O(반응-결과) 시스템의 5주 전환 메커니즘
- **철학**: 매일의 작은 성공이 모여 큰 변화를 만든다
- **기간**: 35일(5주) 습관 형성 사이클
- **추적**: 매일 습관 완료 여부 기록
- **시각화**: 개인 진행도 + 사용자 간 비교

---

## 🎯 주요 기능

### 1. 초기 설정 화면 (첫 방문)
```
[사용자 ID 입력] → [이번 5주 습관 셋 생성] → [습관 리스트 입력]
```
- **사용자 ID**: 고유 식별자 (이메일 또는 닉네임)
- **5주 기간 표시**: 2025.04.08 ~ 2025.05.13 (시작일 기준 자동 계산)
- **습관 추가** (이 5주 동안 함께 추적할 습관들):
  - 입력창 + [추가] 버튼
  - 각 습관 옆에 분류 태그 (S-R / R-O / 혼합)
  - 최소 3개, 최대 10개 추천
  - **예**: 운동 30분, 명상 10분, 독서 20분, 일기, 코딩
- **주의**: 이 5주 동안은 **같은 습관들로 매일 추적**

### 2. 메인 대시보드

#### 2.1 상단 네비게이션 (자동 단계 표시)
```
┌───────────────────────────────────────┐
│ 🧠 중기 (8-35일) [ⓘ] ⚙️ 설정         │
│ "일관성이 가장 중요합니다. 계속하세요!" │
└───────────────────────────────────────┘
```

**자동 판단 로직:**
- **Day 1-7**: 초기 (초기 - 목표-지향 학습)
- **Day 8-35**: 중기 (전환 - 뇌가 적응 중)
- **Day 36+**: 후기 (자동화 - 생각 없이 반복)

**[ⓘ] 정보 버튼**: 현재 단계에 대한 설명 팝업
- 뇌의 변화 (DMS/PFC/DLS 상태)
- 이 시기의 핵심 전략
- 예상되는 진행도

**설정(⚙️)**: 습관 수정, 새 셋 시작 등

**특징:**

- 한 화면: 14개 날짜 (모두 표시)
- 최신 날짜가 좌측
- 오늘: 글로우 효과
- **날짜 라벨**: 세로 회전, 32px 흰색 박스, 명확한 가독성
- 컨테이너 높이: 120px (넉넉함)

```

              0 1 2 3 4 5

●  05/12
  ●  05/11
    ●  05/10
      ●  05/09
        ●  05/08
     ...
    ●  04/08

─ = 그리드 라인
```

**특징:**
- 세로 방향 점 그래프 (선 없음)
- Y축: 날짜 (35개 행)

- 각 점으로 일일 진행도 표시
- barSize=5px (작고 명확함)
- 전체 높이: 560px (35개 행 * 16px)
- 오늘: 진한 파란색 (#3B82F6) + 글로우
- 그리드: 가로만 표시

**색상:**

### 4. 설정 메뉴 (⚙️)
```
┌────────────────────────────────┐
│ ⚙️ 설정                        │
├────────────────────────────────┤
│ 📋 이번 5주 습관 수정/관리     │
│ 🔄 새로운 5주 습관 셋 시작     │
│ 📊 데이터 내보내기             │
│ 👥 "겨루기" 활성화            │
│ 📈 지난 셋 보기 (기록만)       │
│ 🗑️ 모두 초기화                │
└────────────────────────────────┘
```

**상세 기능:**
- **이번 5주 습관 수정**: 각 습관 이름/분류 변경, 삭제 (현재 진행 중인 5주만)
- **새로운 5주 습관 셋 시작**: 
  - 5주 완료 후 새로운 습관들로 새 셋 시작
  - 기존 5주 데이터는 자동 보존
  - "지난 셋 보기"에서 과거 데이터 열람 가능
- **데이터 내보내기**: CSV 다운로드
- **"겨루기" 활성화**: 친구 사용자 ID 입력 → 비교 대시보드 표시
- **지난 셋 보기**: 과거 5주 기간들의 그래프 열람 (읽기 전용)
- **모두 초기화**: 현재 셋 초기화 (경고: 데이터 삭제)

## 🔍 S-R vs R-O 균형 권장사항

### 초기 5주 추천 설정
```
전체 습관: 5-6개

S-R 습관: 2-3개
  - 시간/장소 기반 (아침 7시, 점심 12시 등)
  - 자동화 속도: 빠름 (초기 2주부터 효과)
  - 예: 운동, 명상, 칫솔질

R-O 습관: 3-4개
  - 목표/결과 기반 (지식, 성장 등)
  - 동기 유지 필요 (전체 기간)
  - 예: 독서, 코딩, 언어학습
```

### 진행 과정에서 변화
```
초기 (1-7일):   S-R 30% + R-O 70%
              → 목표 이해와 동기 중심

중기 (8-35일):  S-R 50% + R-O 50%
              → 자동화와 동기 균형

후기 (36-50일): S-R 70% + R-O 30%
              → 자동화된 행동 중심
```
